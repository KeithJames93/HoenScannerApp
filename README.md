# HoenScanner - Flight Information UI

This project is a submission for Task 3, demonstrating the integration of the Skyscanner **Backpack** library in an Android application.

## Features
* **Backpack Integration:** Uses `BpkCardView` and `BpkText` for a native Skyscanner look and feel.
* **Flight Details:** Displays flight number, departure airport/time, and arrival airport/time.
* **Modern Layout:** Built using a vertical `LinearLayout` nested within a `BpkCardView`.

## Tech Stack
* **Language:** Kotlin
* **UI Library:** net.skyscanner.backpack:backpack-android:43.0.0
* **Minimum SDK:** API 33
